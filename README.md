# 📊 E-Commerce Sales Analysis using SQL, Excel & Power BI

## Project Overview
This project analyzes an E-Commerce Sales dataset using SQL, Microsoft Excel, and Power BI to identify sales trends, profit analysis, customer behavior, and business insights.

## Tools Used
- MySQL
- Microsoft Excel
- Power BI

## Project Objectives
- Analyze sales performance
- Identify top-performing products
- Evaluate regional sales
- Analyze customer segments
- Create an interactive dashboard

## Skills Demonstrated
- SQL Queries
- Data Cleaning
- Data Analysis
- Power BI Dashboard
- Excel Dashboard
- Business Intelligence
- Data Visualization

## Key Insights
- Total Sales Analysis
- Total Profit Analysis
- Sales by Category
- Sales by Region
- Top Customers
- Monthly Sales Trend
